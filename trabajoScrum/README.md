# Trabajo Scrum

## Creacion del proyecto

```

Como scrum master ( Otar Kebadze ), me toco crear el proyecto, en el cual estaran todas las tarjetas para que los chicos lleven a cabo sus tareas

```

![proyecto-imagen](/img/proyecto.png)



<h2 align="center">Creacion De Las Tarjetas De Trabajos<h2>

```

Una vez adentro del proyecto, crearemos las tarjetas personales para cada uno, con la descripcion de sus tareas, tambien asisgnandolos

```

![proyecto-imagen](/img/descripcion-tarjeta.png)

<h2 align="center">Issues Y Milestones<h2>

```

Despues de crear todas las tarjetas , se procede a crear las milestones e issues, para que asi se pueda explicar que es lo que debe realizarse en cada tarjeta, y a la vez poder tener una comunicacion clara entre las partes, en caso de haber alguna confusion o en caso de necesitarse una explicacion mas clara

```

![proyecto-imagen](/img/issues.png)
![proyecto-imagen](/img/issueymilestone.png)
